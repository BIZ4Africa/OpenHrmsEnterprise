Enterprise Open HRMS GOSI
=========================
When HR Manager will generate payslip system will automatic calculate GOSI if
GOSI detail exist on employee personal information. And also allows to print
the pdf reports of those Contribution registers.

Installation
------------
- www.odoo.com/documentation/18.0/setup/install.html
- Install our custom addon

Configuration
=============
- Additional configuration not required
