How do I set the company policy?

Go to Settings → Companies, open your company form, and navigate to the Company Policy tab. Use the rich text editor to write and format your policy, then save.

Who can view the company policy?

Any logged-in user with access to the HR Dashboard can view the company policy by clicking the Company Policy button. No special permissions are required beyond basic user access.

Who can edit or update the company policy?

Only users with Administrator access or those with rights to edit the company form in Settings can add or modify the company policy content.

Does this module support multiple companies?

Yes. Each company can have its own separate policy. When a user logs in, the policy displayed is always linked to their currently active company, thanks to the built-in multi-company rule.

Can I format the policy text with images or bullet points?

Yes. The policy field uses Odoo's built-in HTML widget, which provides a full rich text editor. You can add headings, bullet points, bold/italic text, links, and images.

Where does the Company Policy button appear on the dashboard?

The button appears in the left employee profile panel of the HR Dashboard. Clicking it opens a popup dialog displaying the full company policy for the currently logged-in user's company.

What happens if no policy has been set yet?

The popup will open with an empty policy field. No error is shown. The administrator can then navigate to Settings and add the policy content at any time.

Is this module compatible with Odoo 18?

Yes. This module has been fully migrated and tested for Odoo 18. It uses the OWL framework for frontend components and follows Odoo 18 standards for models, views, and assets.
